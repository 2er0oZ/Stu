## 项目亮点

- **MVC 三层架构**：Bean、DAO、Service、Controller 分层清晰
- **Filter 过滤器**：实现登录验证和字符编码统一处理
- **SQL 注入防护**：使用 PreparedStatement 预编译语句
- **JSTL 标签库**：简化 JSP 页面开发
